# **App Name**: MaterialFlow

## Core Features:

- User Authentication: Secure login system with four role-based access levels: Director, Site Manager, Coordinator, and MAPI Store Manager.
- Material Request Management: Submit material requests, track request status, and manage extensions. Auto generate bills for each request.
- Material Receipt Management: Accept material deliveries, verify quantities, and generate receipt bills. Handle discrepancies and return incorrect quantities to the site.
- Inventory Tracking: Monitor material availability across all sites and the central store. Track material status with low stock alerts based on site-specific minimum quantities.
- Invoice Upload: Upload invoices with multiple materials and rates.
- Role-Based Dashboards and Reports: Role-specific dashboards provide a summary of material status, requests, receipts, low stock alerts, and shifting details. Generate Excel reports for detailed analysis.
- Bill Generation AI Review Tool: An AI powered tool to cross-check material bills by cross-examining all related entries in the ledger

## Style Guidelines:

- Primary color: A muted blue (#5DADE2) evoking a sense of trust and reliability, crucial for a supply chain management tool.
- Background color: A very light blue (#EBF4FA), almost white, providing a clean and uncluttered backdrop for the data-rich interface.
- Accent color: A warm orange (#F39C12), contrasting with the blue to highlight key actions and alerts.
- Font pairing: 'Inter' (sans-serif) for both headings and body text, to provide a clear and modern reading experience; 'Source Code Pro' (monospace) for any displayed code snippets.
- Use consistent, clean, and easily recognizable icons to represent different material types and actions.
- A responsive, grid-based layout to ensure data is clearly presented and easily accessible across different screen sizes.
- Subtle animations to indicate loading states and provide feedback on user interactions, enhancing the overall user experience.
